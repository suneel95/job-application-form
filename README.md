# Online Job Application Form

A clean, structured **Online Job Application Form** built using semantic HTML5 and custom CSS3. This project demonstrates a focus on user interface (UI) design, utilizing a warm color palette and an organized multi-column layout for a streamlined user experience.

## ‚ú® Features

* **Organized Sections:** Uses a tabbed "Personal Information" header to group related data logically.
* **Diverse Input Types:**
    * **Text Inputs:** For names and places.
    * **Textareas:** Expandable fields for Current and Permanent addresses.
    * **Date Picker:** Native HTML5 date input for Date of Birth.
    * **Radio Buttons:** Custom-styled selection for gender.
* **Grid-based Layout:** A clean 3-column structure for desktop viewing.
* **Consistent Styling:** Unified design language with rounded corners and high-contrast labels.
